# Array

Może przechowywać różne typy danych

```python
tablica = []  # Tablica jest w tym momencie pusta
tablica.append(5)  # tablica[0]
tablica.append(8)  # tablica[1]
tablica.append(9)  # tablica[2]
print(tablica)  # [5, 8, 9]
```
